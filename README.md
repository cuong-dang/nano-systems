# nano-systems
This repository contains from-scratch implementations of interesting
computing systems (e.g., interpreters, operating systems, databases).
Unlike the [study](https://github.com/cuong-dang/study) repository,
which focuses on exercises and partial implementations, this repo
contains complete, working systems, albeit modest in scope.

- zlox: Bytecode VM for the Lox programming language
- db: Disk-based relational db management system
